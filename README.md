# RNA-seq Integrative Pipeline

## Overview
End-to-end RNA-seq workflow including upstream processing
(alignment and quantification) and downstream differential
expression and multi-omics integration analysis.

## Upstream
- FastQC
- Trimmomatic
- HISAT2
- Samtools
- FeatureCounts

## Downstream
- edgeR / limma-voom
- Multi-omics integration
- Starburst visualization

## Reproducibility
- Logging system
- Tool version tracking
- Input validation
- Session information export
